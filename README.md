## Create

```bash
//new solution
dotnet new sln -o dotnet_minimal_template

//new api
dotnet new webapi -minimal -o api
dotnet sln add ./api/api.csproj

//new mstest
dotnet new mstest -o test
dotnet sln add ./test/test.csproj
```

